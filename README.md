# KrishiBuddy-AI-Farming-Assistant
KrishiBuddy is an AI-powered agriculture support system that helps farmers with crop recommendations, weather insights, soil analysis, and chatbot-based agricultural assistance. Built using Python, ML models, and a fast backend API.
KrishiBuddy is an AI-powered agriculture support platform designed to assist farmers through smart recommendations, soil analysis, weather insights, and an intelligent chatbot.
Built using Python, it uses machine learning models and clean backend architecture to provide reliable agricultural decision support.
